Projet Needleman-Wunsch :

- Pour utiliser ce programme, lancez la commande, dans un terminal : python3 menu.py

- Attention, ce programme necessite la librairie numpy et re.

- Il offre la possibilité à l'utilisateur d'aligner des séquences protéiques et nucléotidiques.

- Il est capable de comparer deux fichiers de séquence, ainsi que de saisir manuellement celles-ci.

- Le résultat est ecrit dans un fichier de sortit "outFileRes.txt"
